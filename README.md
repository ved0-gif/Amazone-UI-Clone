# Amazon UI Clone (HTML & CSS)

This is a frontend UI clone of the Amazon homepage created using only HTML and CSS.

## Features
- Navigation bar
- Search bar
- Product sections
- Hero banner
- Footer
- Layout styling using Flexbox

## Purpose
I built this project to practice real-world website layout design and improve my CSS positioning and alignment skills.

## Tech Used
HTML5
CSS3
